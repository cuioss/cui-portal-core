= portal-core

== What is it?

Provides types that define the non-ui parts of the portal.

== Maven Coordinates

[source,xml]
----
    <dependency>
        <groupId>de.cuioss.portal.core</groupId>
        <artifactId>portal-core</artifactId>
    </dependency>
----
